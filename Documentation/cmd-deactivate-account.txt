gerrit deactivate-account
=========================

NAME
----
gerrit deactivate-account - Deactivate an account.

SYNOPSIS
--------
[verse]
'ssh' -p <port> <host> 'gerrit deactivate-account' \
<USERNAME>

DESCRIPTION
-----------
Deactivates a user by removing his/her SSH keys and setting the
active flag in the database to false.

If the authentication is based on LDAP, the user is not verified
against it but only against the current database. Therefore this
script should be called after the user is removed from LDAP to
ensure access restrictions are working as intended.

ACCESS
------
Caller must be a member of the privileged 'Administrators' group.

SCRIPTING
---------
This command is intended to be used in scripts.

OPTIONS
-------
<USERNAME>::
	Required; SSH username of the user account.

EXAMPLES
--------
Deactivate a user account called `watcher`:

====
	$ ssh -p 29418 review.example.com gerrit deactivate-account watcher
====
