= gerrit index status

== NAME
gerrit index status - Get the online indexer status

== SYNOPSIS
[verse]
--
_ssh_ -p <port> <host> _gerrit index status_ <INDEX>
--

== DESCRIPTION
This command displays the online indexer status.

== ACCESS
Caller must be a member of the privileged 'Administrators' group.

== SCRIPTING
This command is intended to be used in scripts.

== OPTIONS
<INDEX>::
  The secondary index name.
  Currently supported values:
    * changes
    * accounts

== EXAMPLES
Show the status of the online indexer for the 'changes' index:

----
  $ ssh -p 29418 review.example.com gerrit index status changes
----
