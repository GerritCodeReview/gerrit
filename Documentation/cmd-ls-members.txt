gerrit ls-members
=================

NAME
----
gerrit ls-members - List group members

SYNOPSIS
--------
[verse]
'ssh' -p <port> <host> 'gerrit ls-members'
  [--recursive]
  [--project <NAME>]
  [--tree]
  <GROUP>

DESCRIPTION
-----------
Displays the members for a group, if the group is visible to the
calling user.

Listing the members of the 'Registered Users' group may be time
consuming and result in a very large result set.

ACCESS
------
Any user who has configured an SSH key.

SCRIPTING
---------
This command is intended to be used in scripts.

OPTIONS
-------
<GROUP>::
	Required; name of the group for which the members should be
	listed.

--recursive::
-r::
	If specified the included groups will be resolved recursively.

--project::
-p::
	Name of the project for which the link:access-control.html#project_owners[Project Owners]
	group will be resolved. If not specified the 'Project Owners'
	group will not be resolved.

--tree::
-t::
	Displays the group hierarchy in a tree-like format.
	If this option is set the included groups will always be resolved
	recursively.

EXAMPLES
--------

List direct members of the group 'MyCommiters':
=====
	$ ssh -p 29418 review.example.com gerrit ls-members MyCommiters
	Accounts:
	Joe Average <joe.average@example.com> (1000005)
	John Doe <john.doe@example.com> (1000001)

	Included Groups:
	OtherCommitters (10)
=====

List all members of the group 'MyCommiters':
====
	$ ssh -p 29418 review.example.com gerrit ls-members --recursive MyCommiters
	Accounts:
	Joe Average <joe.average@example.com> (1000005)
	Joe Public <joe.public@example.com> (1000003)
	Joe Sixpack <joe.sixpack@example.com> (1000007)
	John Doe <john.doe@example.com> (1000001)

	Included Groups:
	OtherCommitters (10)
	OtherVerifiers (11)
====

List the project owners for the project 'MyProject':
====
	$ ssh -p 29418 review.example.com gerrit ls-members --recursive --project MyProject "'Project Owners'"
	Accounts:
	Joe Average <joe.average@example.com> (1000005)
	John Doe <john.doe@example.com> (1000001)

	Included Groups:
	MyProject_Owners (4)
====
