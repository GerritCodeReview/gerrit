plugin remove
=============

NAME
----
plugin remove - Remove plugins.

plugin rm - Remove plugins.

SYNOPSIS
--------
[verse]
'ssh' -p <port> <host> 'gerrit plugin remove | rm'
  <NAME> ...

DESCRIPTION
-----------
Remove plugins. The plugins will be deleted from the site path's
`plugins` directory.

ACCESS
------
Caller must be a member of the privileged 'Administrators' group.

SCRIPTING
---------
This command is intended to be used in scripts.

OPTIONS
-------
<NAME>::
	Name of the plugin that should be removed.  Multiple names of
	plugins that should be removed may be specified.

EXAMPLES
--------
Remove a plugin:

====
	ssh -p 29418 localhost gerrit plugin remove my-plugin
====
