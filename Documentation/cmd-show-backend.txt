= gerrit show-backend

== NAME
gerrit show-backend - Display SSH backend

== SYNOPSIS
--
'ssh' -p <port> <host> 'gerrit show-backend'
--

== DESCRIPTION
Show used SSH backend

== ACCESS
Caller must be a member of the privileged 'Administrators' group.

== SCRIPTING
Intended for interactive use only.

== EXAMPLES

====
	$ ssh -p 29418 review.example.com gerrit show-backend
	nio2
====
