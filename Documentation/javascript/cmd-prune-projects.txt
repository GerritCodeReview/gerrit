gerrit prune-projects
=========================

NAME
----
gerrit prune-projects - Force deletion of projects in PRUNE status.

SYNOPSIS
--------
[verse]
'ssh' -p <port> <host> 'gerrit prune-projects'

DESCRIPTION
-----------
Forces deletion of projects in PRUNE status independent of prune delay.
It will put these projects out of database and off disk immediately.

ACCESS
------
Caller must be a member of the privileged 'Administrators' group.

SCRIPTING
---------
This command is intended to be used in scripts.

OPTIONS
-------
\--force-prune-right-now::
	Required option to be extra safe.

EXAMPLE
--------

====
	$ ssh -p 29418 review.example.com gerrit prune-projects --force-prune-right-now
====
