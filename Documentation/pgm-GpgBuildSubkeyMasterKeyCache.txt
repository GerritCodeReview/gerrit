= GpgBuildSubkeyMasterKeyCache

== NAME
GpgBuildSubkeyMasterKeyCache - Build map from subkey id to master key id.
This is needed for fast access to the key ring when subkeys are used for
signing.

== SYNOPSIS
[verse]
--
_java_ -jar gerrit.war _GpgBuildSubkeyMasterKeyCache_
  -d <SITE_PATH>
--

== DESCRIPTION
Add subkey -> master keys map as persistent cache to lookup for master
GPG key ring with subkey.

== OPTIONS

-d::
--site-path::
	Location of the gerrit.config file, and all other per-site
	configuration data, supporting libraries and log files.

== CONTEXT
This command can only be run on a server which has direct
access to `All-Users` git repository.

== EXAMPLES
To build GPG subkey cache:

----
	$ java -jar gerrit.war GpgBuildSubkeyMasterKeyCache -d site_path
----
